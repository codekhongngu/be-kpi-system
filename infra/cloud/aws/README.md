# AWS Infrastructure Configuration

This directory contains AWS-specific infrastructure configurations.

## Structure

- `ecs/` - ECS task definitions and service configurations
- `ec2/` - EC2 launch templates and user data scripts
- `rds/` - RDS database configurations
- `s3/` - S3 bucket policies and configurations
- `cloudformation/` - CloudFormation templates
- `scripts/` - AWS CLI scripts for deployment

## Usage

Place your AWS infrastructure configurations here.
